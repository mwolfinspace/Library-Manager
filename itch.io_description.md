## About This Archive

Welcome to Xedryk Archive — a unique digital collection of illustrated stories and artworks. This archive serves as a personal gallery and story repository, featuring:

- **10+ Illustrated Reports** — Each entry contains custom artwork and narrative content

## Features

### 🔍 Advanced Search
Filter reports by title, description, or tags. Supports advanced operators:
- `tag1 tag2` — Stories with either tag
- `+tag1 +tag2` — Stories with BOTH tags
- `-tag1` — Exclude stories with tag1
- `tag1 | tag2` — Stories with tag1 OR tag2

### 🎨 Customization
- **8+ Font Options** — Choose from tech, serif, sans-serif, and more
- **Dark/Light Theme** — Toggle between themes
- **Custom Colors** — Personalize every UI element
- **Keyboard Shortcuts** — Fully rebindable controls

### 📊 Multiple Layouts
- **Grid** — Classic card-based view
- **List** — Compact horizontal layout
- **Compact** — Dense information view
- **Spotlight** — Large featured cards

### ⌨️ Keyboard Shortcuts
- Arrow keys for navigation
- Zoom and pan images
- Toggle favorites and bookmarks
- Quick search access
- Blackout mode for privacy

### 🔧 Settings & Data
- Export/import all settings and progress
- Reset options at various levels
- Persistent local storage

## Download & Install Instructions

This is a **web-based archive** — no installation required!

### How to Run

1. **Extract the ZIP file** to any folder on your computer
2. **Open `homepage.html`** in a modern web browser (Chrome, Firefox, Edge, Safari)
3. That's it! The archive will load directly in your browser

### System Requirements

- **Browser:** Chrome, Firefox, Edge, Safari, or any modern browser
- **Internet:** Not required (fully offline-capable)
- **Storage:** ~50MB free space

### Notes

- All data is stored locally in your browser (localStorage)
- No server or backend required
- Works on Windows, Mac, and Linux
- Recommended: Enable JavaScript for full functionality

### Troubleshooting

- **Images not loading:** Make sure the `img-scr/`, `database/`, and `view/` folders are in the same directory as `homepage.html`
- **Fonts missing:** The archive will use fallback fonts if custom fonts don't load
- **Settings not saving:** Check that your browser allows localStorage
