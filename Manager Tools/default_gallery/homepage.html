🔍 Search Guide

tag1 tag2 Stories with either tag1 OR tag2
+tag1 +tag2 Stories with BOTH tags (required)
-tag1 Exclude stories with tag1
tag1 | tag2 Stories with tag1 OR tag2 (pipe separator)
action +sci-fi -romance Action that is sci-fi but NOT romance
